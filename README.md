# bypass-local-lists

**BypassSec LLC**-ийн боловсруулсан Монголын дотоод кибер аюулгүй байдалд чиглэсэн хэрэглэгчийн нэр, утасны дугаар болон бусад үгсийн wordlist цуглуулга.

Энэхүү репозитор нь Pentest, OSINT, password-spraying, enumeration болон бусад судалгааны зориулалтаар ашиглахад зориулсан, Монгол нэршил, соёлд суурилсан wordlist-үүдийг төвлөрүүлсэн. Мэдээлэл ба wordlist-үүд нь нээлттэй ашиглах боломжтой боловч ашиглахдаа зохих зөвшөөрөл болоод ёс зүйг баримтална уу.
